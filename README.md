# Apollo_SDK_exe
[![Version](https://badge.fury.io/gh/tterb%2FHyde.svg)](https://badge.fury.io/gh/tterb%2FHyde)

It is a program from sdk slamtec developed by c ++.


## Method of use
* status
* version
* move
* home

### Status of Robot
   
```
burl status
```

### Version of Robot
   
```
burl version
```

### Move
   
```
burl move [position X] [position Y]
```

### Rotate
   
```
burl rotate [] [] []
```

### Home
   
```
burl gome
```
